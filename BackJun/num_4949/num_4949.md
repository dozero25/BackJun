1. 조건 추가
- 백준 9012번에서 조건을 추가해서 문제 풀이
- 입력값 `str`이 `.`을 받으면 `while` 종료
- `ch == '(' || ch == '['`이면 해당 값 `push()`
- `ch == ')' || ch == ']'` 일때
  - `stack`이 비어있을때
  - `ch == ')`일 때 `stack`의 맨 위 값이 `(`이 아닐때
  - `ch == ']`일 때 `stack`의 맨 위 값이 `[`이 아닐때
  - `isValid = false`로 변경 후 `break`
  - if문이 실행 되지 않으면 `pop()` 실행
